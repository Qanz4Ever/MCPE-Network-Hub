# 🌐 MCPE Network Hub

A curated web portal for Minecraft: Bedrock Edition server listings — featuring a sleek, modern interface with animated starfield, smooth page transitions, and a responsive layout. Built with HTML, CSS, and vanilla JavaScript. Designed for personal deployment via GitHub Pages.

> ⚠️ **Important**  
> This repository is intended for personal use only. Cloning, redistribution, or reuse without explicit permission is prohibited unless permitted under the terms of the licenses below.

---

## ✨ Features

- 🎨 **Modern Glassmorphic UI** – Frosted glass effect with glowing accents  
- 🌟 **Animated Starfield Background** – Subtle, dynamic stars with twinkling effect  
- 🔄 **Smooth Page Transitions** – Elegant fade animation when navigating  
- 📱 **Fully Responsive** – Optimized for desktop, tablet, and mobile  
- 🎮 **Server Card Display** – Showcases server name, version, game mode, and icon  
- 🔗 **Community Button** – Quick link to external community platform  

---

## 🛠️ Technologies Used

- **HTML5** – Semantic structure  
- **CSS3** – Flexbox, gradients, backdrop-filter, animations  
- **JavaScript (Vanilla)** – Starfield animation, transition handling, dynamic canvas  

---

## 📁 Project Structure

```

.
├── index.html          # Main landing page
├── style.css           # All styles and responsive design
├── script.js           # Starfield animation and navigation logic
├── favicon.png         # Site favicon
└── pteronet-legacy/    # Subpage directory (server details)
└── media/
└── server-icon.png  # Server icon asset

```

---

## 🚀 Deployment

This project is designed to be deployed via **GitHub Pages**.  
Simply push to a repository and enable Pages from the repository settings.

No build steps or dependencies required.
